# QuickPost  

**Платформа для автопостинга в Telegram, VK, OK**  

Простой и удобный сервис для создания и публикации постов. 

## Возможности  

- Создание и редактирование постов  
- Быстрая публикация без лишних шагов
- Удобное управление аккаунтами

## Технологии  

- **Frontend**: React
- **Backend**: Node.js
- **DB**: MySQL
